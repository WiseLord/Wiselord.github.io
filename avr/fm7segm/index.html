Project overview

This project is a simple FM radio tuner with 4-digits LED indication. Up to 50 FM stations can be stored in EEPROM memory of ATmega8 MCU. Also device can show current time and temperature.

Main features:

  • Up to 50 FM stations can be stored in EEPROM.
  • Works with both common anode and common cathode LED indicators, wired as directly to MCU pins as via transistor switches.
  • 12 levels of brightness
  • 16 levels of volume
  • Stations are always saved sorted
  • RTC DS1307 support (DS3231 also works)
  • Temperature measurement with DS18B20 sensors

Supported FM tuners:

  • TEA5767 via I²C bus
  • RDA5807M via I²C bus
  • TUX032 (can be found in some Sony Car Audio) via I²C bus
